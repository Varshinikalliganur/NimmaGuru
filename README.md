# Nimmaguru – Smart Learning Platform 📚

Nimmaguru is an Android-based educational application developed using **Kotlin**, **Jetpack Compose**, and **Firebase**. The app is designed to provide students with an easy and interactive learning experience through courses, videos, authentication, and personalized content.

## 🚀 Features

- User Registration and Login
- Firebase Authentication
- Interactive Home Dashboard
- Course Listing and Details
- Video Learning Module
- User Profile Management
- Settings Screen
- Smooth Navigation using Jetpack Compose
- Modern UI Design
- MVVM Architecture

## 🛠️ Technologies Used

- Kotlin
- Jetpack Compose
- Firebase Authentication
- Firebase Firestore
- Android Studio
- MVVM Architecture
- Navigation Component

The application uses Firebase services:

- Firebase Authentication
- Cloud Firestore Database
- Real-time data management

## 🏗️ Architecture

The project follows the **MVVM (Model–View–ViewModel)** architecture:

- Model → Data classes and Firebase data
- View → Jetpack Compose UI screens
- ViewModel → Business logic and state management

## ⚙️ Installation Steps

1. Clone repository:

```bash
https://github.com/Varshinikalliganur/NimmaGuru/new/main?filename=README.md
```

2. Open Android Studio

3. Open the project folder

4. Connect Firebase project

5. Add `google-services.json`

6. Sync Gradle

7. Run the application
